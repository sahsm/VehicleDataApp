# Vehicle Data Dashboard

This project is an interactive web dashboard built with Streamlit using a vehicle listings dataset.

The app allows users to:
- View a sample of the dataset
- Generate a histogram of vehicle mileage (odometer)
- Generate a scatter plot comparing mileage and price
